Code Examples

v1.0.00

This directory contains several code examples for PIC32 family of microcontrollers. Each directory may contain more than one code example. Each code example directory contains complete source code and MPLAB project files. In addition, it also contains a build.bat file to build the project from the command-line. There is also a build_all.bat file at the top level to build all examples at once from the command-line.

Directory Structure

The Code Examples directory structure consists of multiple high-level directories for each major topic. At the time of publishing this version, there were following major topics:

 

Example Topic/Directory

Description

assembly_examples

MPLAB ASM32 Assembly language examples

c32_examples

MPLAB XC32 syntax/extension specific examples

plib_examples

Peripheral Library usage examples
