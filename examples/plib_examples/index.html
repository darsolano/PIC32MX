Peripheral Library Code Examples

This directory contains PIC32 Peripheral Library code examples for MPLAB XC32 C compiler. This directory may contain more than one code example. Each code example directory contains complete source code and MPLAB project files. In addition, it also contains a build.bat file to build the project from the command-line. There is also a build_all.bat file at the top level to build all examples at once from the command-line. The remainder of this document provides a brief overview of each example

Peripheral Library Examples

Note: These examples use peripheral library binaries distributed with MPLAB XC32. The source files are located in <xc32-install-path>/pic32-libs/include/peripheral and <xc32-install-path>/pic32-libs/peripheral directories. Refer to <xc32-install-path>/docs/pic32_peripheral_libraries.pdf for more information.
